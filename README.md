# Legali-Scan-The-Multimodal-M-A-Due-Diligence-Engine
In corporate Mergers &amp; Acquisitions (M&amp;A), lawyers have to manually review 50,000+ pages of contracts, bank statements, and floor plans. Legali-Scan does this in minutes.

## Stack: Python (Backend), FastAPI, Streamlit (Frontend), Gemini 1.5 Pro API.**

1. Install the dependencies
`pip install requirements.txt`
2. After installation, run this command `streamlit run app.py`
